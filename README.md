## Usage

```python
from gridlab import World


w = World()
w.initialize(builder=factory)

# gridlab.run_demo(w)

while True:
    grid: str = w.to_symbolic_grid()
    desc: str = w.to_text_description()
    action = get_action(state)
    w.step(action)
    if w.is_finished:
        break

```
